GridOptions (api in module ui.grid.grouping )

Description

GridOptions for grouping feature, these are available to be set using the ui-grid gridOptions

Properties

  • enableGrouping

    Enable row grouping for entire grid.
    Defaults to true

  • groupingIndent

    Number of pixels of indent for the icon at each grouping level, wider indents are visually more pleasing, but may result in you having to make the group row header wider
    Defaults to 10

  • groupingRowHeaderWidth

    Width of the grouping header, if your nested grouping is too deep you may need to increase this
    Defaults to 40
