uiGridTreeViewService (service in module ui.grid.treeView )

Description

Services for treeView features

Methods

  • addOrUseState(grid, row, parents)

    If a state already exists for this row with the right parents, use that state, otherwise create a new state for this row and set it's expand/collapse to the same as it's parent.

    Parameters

    • grid – {grid} –

      the grid we're operating on

    • row – {gridRow} –

      the row we want to set

    • parents – {array} –

      an array of the parents this row should have

    Returns

    {undefined} –

    updates the parents array, updates the row to have a treeExpandedState, and updates the grid.treeView.expandedStates

  • setAllNodes(grid, expandedStatesSubset, targetState)

    Works through a subset of grid.treeView.rowExpandedStates, setting all child nodes (and their descendents) of the provided node to the given state.

    Calls itself recursively on all nodes so as to achieve this.

    Parameters

    • grid – {Grid} –

      the grid we're operating on (so we can raise events)

    • expandedStatesSubset – {object} –

      the portion of the tree that we want to update

    • targetState – {string} –

      the state we want to set it to

  • setCurrentState(parents)

  • toggleRowTreeViewState(grid, row)

    Toggles the expand or collapse state of this grouped row. If the row isn't a groupHeader, does nothing.

    Parameters

    • grid – {Grid} –

      grid object

    • row – {GridRow} –

      the row we want to toggle

  • treeRows(renderableRows)

    The rowProcessor that adds the nodes to the tree, and sets the visible state of each row based on it's parent state

    Assumes it is always called after the sorting processor

    Processes all the rows in order, setting the group level based on the $$treeLevel in the associated entity, and setting the visible state based on the parent's state.

    Calculates the deepest level of tree whilst it goes, and updates that so that the header column can be correctly sized.

    Parameters

    • renderableRows – {array} –

      the rows we want to process, usually the output from the previous rowProcessor

    Returns

    {array} –

    the updated rows, including our new group rows
