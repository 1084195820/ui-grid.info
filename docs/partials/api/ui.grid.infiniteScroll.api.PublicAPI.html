PublicAPI (api in module ui.grid.infiniteScroll )

Description

Public API for infinite scroll feature

Methods

  • dataLoaded(noMoreDataTop, noMoreDataBottom)

    Call this function when you have loaded the additional data requested. You can set noMoreDataTop or noMoreDataBottom to indicate that we've reached the end of your data set, we won't fire any more events for scroll in that direction. See infinite_scroll tutorial for example of usage

    Parameters

    • noMoreDataTop – {boolean} –

      flag that there are no more pages upwards, so don't fire any more infinite scroll events upward

    • noMoreDataBottom – {boolean} –

      flag that there are no more pages downwards, so don't fire any more infinite scroll events downward

Events

  • needLoadMoreData

    This event fires when scroll reaches bottom percentage of grid and needs to load data

  • needLoadMoreDataTop

    This event fires when scroll reaches top percentage of grid and needs to load data
