GridOptions (api in module ui.grid.treeView )

Description

GridOptions for treeView feature, these are available to be set using the ui-grid gridOptions

Properties

  • enableTreeView

    Enable row tree view for entire grid.
    Defaults to true

  • treeViewIndent

    Number of pixels of indent for the icon at each treeView level, wider indents are visually more pleasing, but will make the tree view row header wider
    Defaults to 10

  • treeViewRowHeaderBaseWidth

    Base width of the treeView header, provides for a single level of tree. This is incremented by treeViewIndent for each extra level
    Defaults to 30
